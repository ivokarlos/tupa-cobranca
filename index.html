Como obter os tokens
  1. Acesse console.cloud.google.com
  2. APIs e serviços → Credenciais → edite "Tupã Cobranças"
  3. Acesse developers.google.com/oauthplayground
  4. Engrenagem ⚙️ → marque "Use your own OAuth credentials"
  5. Cole Client ID e Secret → feche
  6. No campo de escopos cole: https://www.googleapis.com/auth/gmail.send
  7. Clique "Authorize APIs" → faça login → autorize
  8. Clique "Exchange authorization code for tokens"
  9. Copie o access_token e refresh_token e cole acima
